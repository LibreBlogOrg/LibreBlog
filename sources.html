Sources Import from RIS File
Support notes
On this page, you can import and delete references. Please note that it is not possible to edit the content of imported references. There are several reference management softwares that are very useful for this task (Zotero, JabRef, etc.).

You can import the same references several times and they will not be inserted twice (as long as they have the same ID or the same type, title, author and year of publication).

P.S. 1: RIS is one of the file formats most commonly used by citation managers such as Mendeley and Zotero to export and import references.

P.S. 2: The "Enable Twig inside articles?" checkbox must be active in Settings for you to be able to insert references into articles.
